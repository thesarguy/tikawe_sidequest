# tikawe-test
sidequest projekti
idea että on sivu, jossa on iso kasa eri ehdotuksia "side-quest" tyylisille jutuille, eli jos on tylsää niin voi etsiä ideoita sille, mitä tehdä. Esim. jotain projekti-ideoita, tavoitteita, uuden taidon oppiminen. Esim "opettele ratkaisemaan rubiikin kuutio", ja sitten olisi siihen ohjeita jne.
Sidequestissa lukee tehtävän kuvaus, mahdolliset ohjeet sekä arvioitu kesto tai vaikeustaso.
Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
Käyttäjä pystyy lisäämään sidequesteja ja muokkaamaan ja poistamaan niitä.
Käyttäjä näkee sovellukseen lisätyt sidequestit, ja kuka ne on lisännyt.
Käyttäjä pystyy etsimään sidequesteja hakusanalla tai esimerkiksi keston, tai muiden kriteerien perusteella.
Käyttäjäsivu näyttää, montako sidequestia käyttäjä on lisännyt ja suorittanut sekä listan käyttäjän lisäämistä sidequesteista.
Käyttäjä pystyy valitsemaan sidequestille yhden tai useamman luokittelun (esim. ulkona, opiskelu, lyhyt, haastava).
Käyttäjä pystyy merkitsemään sidequestin suoritetuksi. Sidequestista näytetään, kuinka moni käyttäjä on suorittanut sen sekä lista suorituksista.
